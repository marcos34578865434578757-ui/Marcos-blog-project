## 这个项目想解决什么

我想把个人博客从“只能手写 Markdown 再手动部署”的状态，推进到“可以在后台完成写作、预览、上传图片和发布”的状态。这样内容生产会更稳定，也更适合后续继续扩展。

## 现在已经做到的部分

- 支持在后台创建和编辑草稿
- 支持导入 Markdown 与 zip 包
- 支持图片上传、封面设置和发布到 GitHub
- 支持在前台按分类归档展示文章

## 下一步重点

接下来更想继续打磨的是内容组织能力，比如分类导航、相关文章关系，以及更舒服的写作体验。等这些能力稳定下来，这套站点就能真正成为长期使用的个人内容系统。
